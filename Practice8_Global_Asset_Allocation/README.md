# Practice 8 — Global Asset Allocation

Strategic asset allocation exercise modeled on a long-only sovereign fund (Norwegian Oil Fund-style), designing 2024 allocation targets across Real Estate, Fixed Income, Large-Cap, and Small-Cap equities. Builds a constrained Markowitz optimal portfolio and compares it against an initial proposal.

**Contents**
- `Practice8_final.mlx` — final MATLAB code
- `Practice8_2024.pdf` — assignment/answers
- `Practice 8 - Presentation.pptx` — group presentation
- `Question1Data2.xlsx`, `Question3Data.xlsx` — source data
