# Practice 4 — Financial Advisor & Client Proposal

Role-play exercise: half the class acts as Financial Advisors, half as clients. As the Financial Advisor, this project develops a Strategic Asset Allocation proposal tailored to client "Ms. Yanny" (30, Chicago-based, working at an energy company), based on her income, expenses, and risk profile.

**Contents**
- `Practice4_2024_Directions.pdf` — assignment brief and client profile
- `p4.mlx` — final MATLAB code
- `Practice4SHEET.xlsx` — supporting spreadsheet
- `Financial Advisor Consulting Agreement.pdf` — consulting agreement document
- `USA inflation rate.csv`, `qqq.csv`, `gld.csv` — source data
