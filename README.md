# Asset Valuation and Selection

Coursework and group projects for the Asset Valuation and Selection course (UC3M). Each folder below is a self-contained practice: MATLAB Live Script (.mlx) code, source data, and the final presentation/answers submitted for grading.

## Projects

| # | Project | Description |
|---|---------|-------------|
| 00 | [Intro Example](./Practice00_Intro_Example) | Worked example of a long-short commodity investment strategy in MATLAB, based on Miffre (2016), used as a template before the first graded practice. |
| 0 | [MATLAB Introduction](./Practice0_MATLAB_Intro) | Warm-up exercises covering MATLAB fundamentals: computing excess returns and synchronizing/cleaning time series with missing data (BTC and CPI datasets). |
| 1 | [BTC, Inflation & Stock Returns](./Practice1_BTC_Inflation_StockReturns) | Regression analysis of the relationship between Bitcoin, inflation, and equity/index returns (SP500, Nasdaq, IBEX35, Nikkei 225, gold, and others). |
| 2 | [Risk Factors & CAPM](./Practice2_RiskFactors_CAPM) | Single-factor regression model (CEI risk factor) applied to stock returns: significance testing, residual diagnostics, and CAPM-based expected return estimation. |
| 3 | [Portfolio Performance Evaluation](./Practice3_Portfolio_Performance_Evaluation) | Evaluation of portfolio performance metrics, referencing Frazzini et al. (2018) "Buffett's Alpha," using Berkshire Hathaway, FTSE World, and Nasdaq 100 data. |
| 4 | [Financial Advisor & Client Proposal](./Practice4_FinancialAdvisor_ClientProposal) | Role-play exercise: building a Strategic Asset Allocation proposal for a fictional client (Ms. Yanny) based on her financial profile and risk constraints. |
| 5 | [Risk Parity vs. Markowitz vs. Equally Weighted](./Practice5_RiskParity_vs_Markowitz_vs_EquallyWeighted) | Comparison of three portfolio construction approaches: Risk Parity, Markowitz mean-variance optimization, and Equally Weighted, on robustness and performance. |
| 6 | [Inflation & Multi-Asset ETFs](./Practice6_Inflation_MultiAsset_ETFs) | Analysis of how inflation (CPI) affects a multi-asset portfolio of stock, bond, and commodity ETFs (plus BTC) using 10+ years of monthly data. |
| 7 | [BMW DCF Valuation - Monte Carlo](./Practice7_BMW_DCF_Valuation_MonteCarlo) | Discounted Cash Flow valuation of BMW under a conservative scenario, using a Monte Carlo simulation to estimate fair value and CAPM/WACC parameters. |
| 8 | [Global Asset Allocation](./Practice8_Global_Asset_Allocation) | Strategic asset allocation exercise for a long-only fund (modeled on the Norwegian Oil Fund): building a constrained Markowitz portfolio across Real Estate, Fixed Income, and Large/Small-Cap equities. |
| 9 | [Market Timing](./Practice9_MarketTiming) | Study of market timing strategies applied to a Dow Jones-based portfolio. |
| 10 | [Integrated Value Creation / ESG](./Practice10_Integrated_Value_Creation_ESG) | Sustainable finance case study on Fund Sustainability Requirements and SFDR (Sustainable Finance Disclosure Regulation) disclosures. |

## Structure

Each project folder typically contains:
- .mlx - MATLAB Live Script with the analysis/code
- Data files (.xlsx, .csv, .mat) - source data used in the analysis
- Presentation/Answers (.pptx, .pdf, .docx) - final deliverable submitted for the practice
