- `Practice0_ExcessReturns_2024.mlx` — computing excess returns
- `Practice0_MissingData_Synch_2024.mlx` — handling missing data and synchronizing time series
- `BTC_daily.xlsx`, `BTC_monthly.xlsx`, `CPI_SP.xlsx`, `Practice0Data.mat` — source data
