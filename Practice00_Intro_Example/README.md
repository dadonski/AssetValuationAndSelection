# Practice 00 — Intro Example: Long-Short Commodity Investing

Worked example used as a template before the graded practices begin. It walks through building a long-short commodity investment strategy in MATLAB, based on the literature review by Joëlle Miffre, "Long-short commodity investing: A review of the literature" (Journal of Commodity Markets, 2016).

**Contents**
- `Individual_Project_example_longshort_2021.mlx` — MATLAB Live Script implementing the long-short strategy
- `JCM_Long-short-investment_20161.pdf` — reference paper (Miffre, 2016)
- `workspace.mat` — saved MATLAB workspace/data
